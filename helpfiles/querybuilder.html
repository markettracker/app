Writing search queries

A search query is a combination of one or more conditions

A condition consists of 3 parts: [FIELD NAME] OPERATOR <VALUE>
E.g. 
[Durability] > 60
Multiple such conditions can be combined by using AND / OR connectors.
E.g. 
[Durability] > 60 AND [Momentum] > 50
Note: you can write complex queries by grouping multiple such queries using brackets. ( )
E.g. 
( [Durability] > 60 AND [Momentum] > 50 ) OR
( [Momentum] > 40 AND [Momentum] < 70 )
